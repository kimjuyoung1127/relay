# docs/internal/daily/ 컨벤션

## 파일명 규칙
- 폴더: `MM-DD/`
- 파일: `module-{모듈슬러그}.md`
- 예: `03-25/module-docs-governance.md`

## 규칙
- 작업 완료 시 해당 날짜 폴더에 로그 작성
- 주간 롤업(`docs/internal/weekly/`)으로 집계

## 주의
- 이 폴더 문서는 당시 시점 기록
- 현재 코드/문서 상태 판단에는 사용하지 않음
- 현재 기준: `docs/status/PROJECT-STATUS.md`, `docs/status/EXECUTION-CHECKLIST.md`
